Web and Programming Journey

I started coding in 2024, and it was a completely new and unfamiliar experience for me. At first, I felt like I was falling behind compared to my classmates. During the first semester, learning HTML wasn't too bad-I actually enjoyed it. But when we were introduced to CSS, it felt overwhelming. Styling a page the way I wanted seemed to take an eternity, and at the time, I thought CSS was the worst thing to code. However, I was completely wrong. After a few weeks of consistent practice, both my HTML and CSS skills improved significantly. I began to feel more confident and thought the second semester would be a smooth ride. But then came JavaScript. For me, JavaScript is incredibly confusing-it feels like a mental puzzle. One small mistake can break the whole website, and that pressure makes it frustrating. I've learned the basics, but I still struggle with it. That's why, to this day, I consider JavaScript my worst enemy when it comes to coding.

Looking back at the beginning of the school year, I'm really proud of how far I've come on my coding journey. I went from not understanding anything to now being comfortable with both HTML and CSS. Will I ever code again after this? Yes, if I want to do something to keep me busy for a few months.
